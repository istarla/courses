## Flex Container Properties (Part 1)

Yesterday, we introduced Flexbox and its core concepts. Today, we'll
start exploring the properties that can be applied to the
**flex container** to control the layout of its flex items.
We'll cover flex-direction, flex-wrap, the
flex-flow shorthand, and justify-content.

## 1. flex-direction


The `flex-direction` property establishes the main axis, thus
defining the direction flex items are placed in the flex container.

Possible values:

* **`row`** (default): Flex items are laid out
  horizontally, from left to right (in LTR languages). The main axis is
  horizontal.
* **`row-reverse`**: Flex items are laid out horizontally, from right to left.
* **`column`**: Flex items are laid out vertically, from top to bottom. The main
  axis is vertical.
* **`column-reverse`**: Flex items are laid out vertically, from bottom to top.

      <div class="flex-container fd-row">
        <div class="flex-item">1</div>
        <div class="flex-item">2</div>
        <div class="flex-item">3</div>
      </div>
      <p>
        <strong>Example: <code>flex-direction: row-reverse;</code></strong>
      </p>
      <div class="flex-container fd-row-reverse">
        <div class="flex-item">1</div>
        <div class="flex-item">2</div>
        <div class="flex-item">3</div>
      </div>
      <p>
        <strong>Example: <code>flex-direction: column;</code></strong>
      </p>
      <div class="flex-container fd-column">
        <div class="flex-item">1</div>
        <div class="flex-item">2</div>
        <div class="flex-item">3</div>
      </div>
      <p>
        <strong>Example: <code>flex-direction: column-reverse;</code></strong>
      </p>
      <div class="flex-container fd-column-reverse">
        <div class="flex-item">1</div>
        <div class="flex-item">2</div>
        <div class="flex-item">3</div>
      </div>

```css
.container-row {
  display: flex;
  flex-direction: row; /* Main axis is horizontal, left to right */
}
.container-column-reverse {
  display: flex;
  flex-direction: column-reverse; /* Main axis is vertical, bottom to top */
}
```

## 2. flex-wrap


By default, flex items try to fit onto one line. The
`flex-wrap` property controls whether the flex container
should wrap its items onto multiple lines if they exceed the container's
width (for rows) or height (for columns).

Possible values:

* **nowrap** (default): Flex items will all be
  on one line. They may shrink or overflow the container if they don't
  fit.
* **wrap**: Flex items will wrap onto multiple lines if necessary, from top to
  bottom.
* **wrap-reverse**: Flex items will wrap onto multiple lines if necessary, from bottom
  to top.

           <p>
        <strong
          >Example: <code>flex-wrap: nowrap;</code> (Default) - Container width
          300px, items 60px wide each</strong>
        >
      </p>
      <div class="flex-container fw-nowrap">

        <div class="flex-item" style="width: 60px">2</div>
        <div class="flex-item" style="width: 60px">3</div>
        <div class="flex-item" style="width: 60px">4</div>
        <div class="flex-item" style="width: 60px">5</div>
      </div>
      <p>
        <strong
          >Example: <code>flex-wrap: wrap;</code> - Container width
          250px</strong>
        >
      </p>
      <div class="flex-container fw-wrap">
        <div class="flex-item" style="width: 60px">1</div>
        <div class="flex-item" style="width: 60px">2</div>
        <div class="flex-item" style="width: 60px">3</div>
        <div class="flex-item" style="width: 60px">4</div>
        <div class="flex-item" style="width: 60px">5</div>
      </div>
      <p>
        <strong
          >Example: <code>flex-wrap: wrap-reverse;</code> - Container width
          250px</strong>
        >
      </p>
      <div class="flex-container fw-wrap-reverse">
        <div class="flex-item" style="width: 60px">1</div>
        <div class="flex-item" style="width: 60px">2</div>

        <div class="flex-item" style="width: 60px">4</div>

```css
.container-wrap {
  display: flex;
  width: 300px; /* To demonstrate wrapping with fixed-width items */
  flex-wrap: wrap;
}
.flex-item {
  width: 100px; /* If container-wrap is 250px wide, 3rd item will wrap */
}
```

## 3. flex-flow (Shorthand)

The `flex-flow` property is a shorthand for setting both
`flex-direction` and `flex-wrap` properties in a
single declaration.

**Syntax:**
`flex-flow: <flex-direction> || <flex-wrap>;`

The default value is `row nowrap`.

      <div
        class="flex-container"
        style="flex-flow: column wrap; height: 150px; width: 200px"
      >
        <div class="flex-item" style="height: 40px">1</div>
        <div class="flex-item" style="height: 40px">2</div>
        <div class="flex-item" style="height: 40px">3</div>
        <div class="flex-item" style="height: 40px">4</div>
        <div class="flex-item" style="height: 40px">5</div>
      </div>

```css
.container-shorthand {
  display: flex;
  flex-flow: column wrap; /* Main axis vertical, wraps items if container height is insufficient */
}
.another-container {
  display: flex;
  flex-flow: row-reverse wrap-reverse;
}
```

## 4. justify-content

The justify-content property defines how flex items are
aligned along the **main axis** of the flex container. It
helps distribute extra free space left over when all the flex items on a
line are inflexible, or are flexible but have reached their maximum
size.

Possible values:

* **`flex-start`** (default): Items are packed
  toward the start of the main axis.
* **`flex-end`**: Items are packed toward the end of the main axis.
* **`center`**: Items are centered along the main axis.
* **`space-between`**: Items are evenly distributed in the line; first item is on the
  start line, last item on the end line. Space is only between items.
* **`space-around`**: Items are evenly distributed in the line with equal space around
  them. Note that visually the spaces aren't equal because each item
  gets space on both sides, so the space between two items is double the
  space before the first and after the last.
* **`space-evenly`**: Items are distributed so that the spacing between any two items
  (and the space to the edges) is equal.

      <div class="flex-container jc-flex-start">
        <div class="flex-item">1</div>
        <div class="flex-item">2</div>
        <div class="flex-item">3</div>
      </div>
      <p>
        <strong>Example: <code>justify-content: flex-end;</code></strong>
      </p>
      <div class="flex-container jc-flex-end">
        <div class="flex-item">1</div>
        <div class="flex-item">2</div>
        <div class="flex-item">3</div>
      </div>
      <p>
        <strong>Example: <code>justify-content: center;</code></strong>
      </p>
      <div class="flex-container jc-center">
        <div class="flex-item">1</div>
        <div class="flex-item">2</div>
        <div class="flex-item">3</div>
      </div>
      <p>
        <strong>Example: <code>justify-content: space-between;</code></strong>
      </p>
      <div class="flex-container jc-space-between">
        <div class="flex-item">1</div>
        <div class="flex-item">2</div>
        <div class="flex-item">3</div>
      </div>
      <p>
        <strong>Example: <code>justify-content: space-around;</code></strong>
      </p>
      <div class="flex-container jc-space-around">
        <div class="flex-item">1</div>
        <div class="flex-item">2</div>
        <div class="flex-item">3</div>
      </div>
      <p>
        <strong>Example: <code>justify-content: space-evenly;</code></strong>
      </p>
      <div class="flex-container jc-space-evenly">
        <div class="flex-item">1</div>
        <div class="flex-item">2</div>
        <div class="flex-item">3</div>
      </div>

```css
.nav-menu {
  display: flex;
  flex-direction: row; /* Main axis is horizontal */
  justify-content: space-around; /* Distribute menu items with space around them */
  list-style: none;
  padding: 0;
}
.vertical-center-column {
  display: flex;
  flex-direction: column; /* Main axis is vertical */
  justify-content: center; /* Centers items vertically in this case */
  height: 300px; /* Container needs a defined height for vertical centering */
}
```

The effect of `justify-content` is most apparent when there's
extra space along the main axis. If items fill the entire main axis
(e.g., due to `flex-grow` or their natural size), some
`justify-content` values might have no visible effect.
