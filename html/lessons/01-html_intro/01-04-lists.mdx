# HTML Lists

## Introduction to Lists


Lists are fundamental for organizing information on web pages. They
allow you to group related items, making content clearer and more
scannable for users. HTML provides three main types of lists: unordered
lists, ordered lists, and definition lists.

## Unordered Lists (`<ul>`)


Unordered lists are used when the order of the items does not matter.
Each item in the list is typically marked with a bullet point (though
this can be changed with CSS).

*Sample Code*
```html
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```
**Output:**
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

Note:

* The entire list is wrapped in a `<ul>` (unordered
  list) tag.
* Each item within the list is wrapped in an
  `<li>` (list item) tag.

*Sample Code*
```html
<ul>
  <li>Apples</li>
  <li>Oranges</li>
  <li>Bananas</li>
</ul>
```

**Rendered Output:**

* Apples
* Oranges
* Bananas

## Ordered Lists (`<ol>`)


Ordered lists are used when the sequence of the items is important. Each
item is typically marked with a number or letter.

* The entire list is wrapped in an `<ol>` (ordered
  list) tag.
* Each item within the list is also wrapped in an
  `<li>` (list item) tag.

*Sample Code*
```html
<ol>
  <li>First Step: Gather ingredients</li>
  <li>Second Step: Mix ingredients</li>
  <li>Third Step: Bake</li>
</ol>
```

**Rendered Output:**

1. First Step: Gather ingredients
2. Second Step: Mix ingredients
3. Third Step: Bake

You can change the numbering type using the `type` attribute
(e.g., `type="A"` for uppercase letters,
`type="a"` for lowercase, `type="I"` for uppercase
Roman numerals, `type="i"` for lowercase) and the starting
number using the `start` attribute (e.g.,
`start="5"`).

## Definition Lists (`<dl>`)


Definition lists (also sometimes called description lists) are used to
present items and their corresponding descriptions, like terms and
definitions in a glossary or key-value pairs.

* The entire list is wrapped in a `<dl>` (definition
  list) tag.
* Each term is wrapped in a `<dt>` (definition term)
  tag.
* Each description/definition associated with a term is wrapped in a
  `<dd>` (definition description) tag.
* You can have multiple `<dd>`s for one
  `<dt>`, or multiple `<dt>`s
  associated with one or more `<dd>`s (though the
  former is more common).

*Sample Code*
```html
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language - used for structuring web content.</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets - used for styling web content.</dd>
  <dd>It makes websites look good!</dd>
</dl>
```

**Rendered Output:**

HTML
:   HyperText Markup Language - used for structuring web content.

CSS
:   Cascading Style Sheets - used for styling web content.
:   It makes websites look good!

## Nesting Lists


You can place lists inside other list items (`<li>`) to
create nested or hierarchical structures. This is common for outlines,
navigation menus, or detailed steps.

## Examples


### Example 1: Simple Shopping List (Unordered)
*Sample Code*
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Shopping List</title>
</head>
<body>
    <h1>Grocery Shopping List</h1>
    <ul>
        <li>Milk</li>
        <li>Bread</li>
        <li>Eggs</li>
        <li>Cheese</li>
    </ul>
</body>
</html>
```
**Output:**
<>
<h1>Grocery Shopping List</h1>
    <ul>
        <li>Milk</li>
        <li>Bread</li>
        <li>Eggs</li>
        <li>Cheese</li>
    </ul>
</>

### Example 2: Recipe Steps (Ordered with Attributes)

*Sample Code*
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Recipe Steps</title>
</head>
<body>
    <h1>How to Make Tea</h1>
    <ol type="1" start="1"> <!-- Default numbering, explicitly starting at 1 -->
        <li>Boil water.</li>
        <li>Place tea bag in cup.</li>
        <li>Pour boiling water into cup.</li>
        <li>Steep for 3-5 minutes.</li>
        <li>Remove tea bag and enjoy.</li>
    </ol>

    <h2>Alternative Options (using letters)</h2>
     <ol type="a">
        <li>Add sugar or honey.</li>
        <li>Add milk or lemon.</li>
    </ol>
</body>
</html>
```

  3. Text Formatting
* CSS Styling
